# Gradually Deprecate Codex Naming

`Codex` is no longer the product language for the STS2 reference area; the canonical service term is the game-localized **백과사전** for **Compendium**. New UI copy, documentation, and component names must avoid `Codex`, while existing code namespaces and routes can be migrated gradually with compatibility preserved because a full immediate rename would create unnecessary routing and regression risk.
