# Gradually Deprecate Codex Naming

`Codex` is no longer the product language for the STS2 reference area; the canonical service term is the game-localized **백과사전** for **Compendium**. New UI copy, documentation, component names, and canonical URLs must avoid `Codex`; `/compendium` is canonical and existing `/codex` routes are compatibility redirects during migration. Existing code namespaces can be migrated gradually because a full immediate rename would create unnecessary routing and regression risk.
