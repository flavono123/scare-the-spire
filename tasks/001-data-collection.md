# Task 001: 데이터 수집

## 설명
슬레이 더 스파이어 전체 카드/유물/변경이력 데이터를 수집하여 JSON으로 정리.

## 작업 항목
- [ ] 전체 카드 데이터 수집 (~200장, 4캐릭터 + 무색 + 저주)
- [ ] 전체 유물 데이터 수집 (~180개)
- [ ] 한국어 공식 번역 이름 확인
- [ ] 변경 이력 수집 (위키 Update History 섹션 기반)
- [ ] `data/cards.json`, `data/relics.json`, `data/changes.json` 작성
- [ ] 이야기 큐레이션 (`data/stories.json`)

## 완료 기준
- 모든 카드/유물이 포함된 JSON 파일
- 위키에서 확인 가능한 변경 이력 포함
