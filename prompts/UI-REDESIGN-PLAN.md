# UI/UX 대개편 계획서

> 2026-04-13 작성. 인게임 코덱스 스타일로 전체 사이트를 통합하는 리디자인.

## 1. 현황 문제

### 1.1 두 개의 분리된 세계
- `(main)` 레이아웃: 상단 네비 + CSS variable 배경 + "슬서운이야기" 브랜딩
- `(codex)` 레이아웃: 네비 없음(뒤로가기 버튼만) + 하드코딩 배경(`#1a1a2e`/`#0a0a0f`) + "Spire Codex" 브랜딩
- 배경색, 폰트 사용법, 레이아웃 패턴이 전부 다름

### 1.2 네비게이션 단절
- 코덱스 내 엔티티 타입 간 이동: 매번 인덱스로 돌아가야 함
- 백과사전 드롭다운에 몬스터/이벤트/인카운터 누락
- STS1↔STS2 전환 경로 없음 (URL 직접 입력 외에)

### 1.3 정체성 혼재
- 페이지 title 3종 혼용: "Spire Codex" / "슬서운 이야기" / "슬서운 카드 도서관"
- 홈(`/`)이 STS1 이야기 피드인데, 핵심 기능은 STS2 패치노트

## 2. 디자인 방향

**인게임 코덱스 스타일**: STS2 게임 내 백과사전 UI를 웹으로 재해석

### 2.1 핵심 원칙
- **게임 프레임**: 전체 사이트를 게임 UI 창처럼 감싸는 외곽 프레임
- **한국어 우선**: "Spire Codex" → "백과사전", 모든 UI 텍스트 한국어
- **통합 네비게이션**: 모든 페이지에서 일관된 상단 탭바
- **STS1/STS2 전환**: 게임 데스크톱 아이콘으로 전환

### 2.2 브랜딩 변경
| 현재 | 변경 후 |
|------|---------|
| "Spire Codex" | **백과사전** |
| "슬서운 이야기" (사이트명) | **슬서운이야기** (유지, 통합 브랜드) |
| page title 혼용 | `{페이지명} — 슬서운이야기` 통일 |

## 3. 정보 구조 (IA)

**슬서운이야기** = 브랜드/서비스명 (내비 항목이 아님, 모든 페이지에서 브랜드로 노출)

```
슬서운이야기 (브랜드, 좌하단 등에 항상 노출)
│
├── /                 이야기 (랜딩/인덱스, STS1+2 통합 콘텐츠)
│                     - STS1→STS2 변경 이야기 등 크로스게임 포함
│
├── /patches          슬서운변경 (주력 서비스)
│   └── /patches/[v]  패치 상세 (rich) → 백과 엔티티로 연결
│
├── /codex            백과사전 (STS2 엔티티 허브)
│   ├── /codex/cards       카드 (612)
│   ├── /codex/relics      유물 (314)
│   ├── /codex/potions     포션 (63)
│   ├── /codex/powers      파워 (238)
│   ├── /codex/enchantments 인챈트 (22)
│   ├── /codex/monsters    몬스터 (111)
│   ├── /codex/events      이벤트 (57)
│   ├── /codex/encounters  인카운터
│   └── /codex/ancients    에인션트 (8)
│   (각 엔티티별 하위 기능 강화 예정)
│
└── /sts1             STS1 레거시 (확장 없음)
    ├── 카드 탐색
    ├── 유물 탐색
    └── 포션 탐색
```

내비게이션에서 게임 아이콘(앱 아이콘)은 STS2/STS1 섹션으로의 분기점.
이야기(/)는 어느 게임에도 속하지 않는 메인 랜딩.

## 4. 레이아웃 구조

### 4.1 통합 셸

모든 페이지를 감싸는 단일 레이아웃 (상단 네비게이션 통일):

```
┌─ Top Bar ──────────────────────────────────────────────┐
│ 슬서운이야기  [STS2앱아이콘] [STS1앱아이콘]  ...섹션들  │
└────────────────────────────────────────────────────────┘
┌─ Content Area ─────────────────────────────────────────┐
│                                                        │
│  (각 섹션의 콘텐츠)                                    │
│                                                        │
└────────────────────────────────────────────────────────┘
```

### 4.2 Top Bar — 게임 아이콘은 분기(내비), 토글 아님

```
┌──────────────────────────────────────────────────────────────┐
│ 슬서운이야기  [STS2🃏] 변경 · 백과 · 이야기  [STS1🃏] 이야기 │
└──────────────────────────────────────────────────────────────┘
```

- STS1/STS2 게임 아이콘(앱 아이콘 에셋)은 **분기 진입점**
  - STS2 아이콘 옆: STS2 전용 섹션 (변경, 백과, 이야기)
  - STS1 아이콘 옆: STS1 전용 섹션 (이야기, 카드, 유물, 포션)
- 토글/전역 모드 전환 아님 — 각 게임은 독립된 기능셋
- STS2가 주력이므로 시각적으로 더 prominent
- 활성 섹션: gold 강조 + 하단 indicator
- 모바일: 햄버거 또는 하단 탭바

### 4.3 백과사전 내부 레이아웃

```
┌─ Category Tabs (인게임 아이콘) ──────────────────────┐
│ 🗡카드  🏺유물  🧪포션  ⚡파워  ✨인챈  👹몬스터...│
├───────────┬──────────────────────────────────────────┤
│           │                                          │
│  목록     │  상세 패널                               │
│  (스크롤) │  (선택한 항목의 인게임 스타일 상세)      │
│           │                                          │
│  필터:    │  ┌──────────────────────┐                │
│  캐릭터   │  │ [카드 이미지]        │                │
│  희귀도   │  │ 이름 / 코스트        │                │
│  검색     │  │ 설명                 │                │
│           │  │ 키워드               │                │
│           │  └──────────────────────┘                │
│           │                                          │
└───────────┴──────────────────────────────────────────┘
```

- **카테고리 탭**: 인게임 아이콘 사용, 가로 스크롤
- **목록 패널**: 현재 사이드바와 유사하되 통일된 디자인
- **상세 패널**: 인게임 툴팁/카드 뷰 스타일

## 5. 시각 디자인

### 5.1 통합 색상 시스템

현재 하드코딩된 hex를 CSS variable로 통합:

```css
:root {
  --frame-bg: #0a0a0f;       /* 게임 프레임 배경 */
  --panel-bg: #12121a;       /* 패널/카드 배경 */
  --surface-bg: #1a1a2e;     /* 콘텐츠 영역 배경 */
  --sidebar-bg: #16162a;     /* 사이드바/툴바 배경 */
  --border-gold: #d4a843;    /* 골드 테두리 */
  --border-dim: rgba(120, 80, 20, 0.2); /* 은은한 테두리 */
  --text-gold: #d4a843;      /* 골드 텍스트 */
  --text-primary: #e5e5e5;   /* 기본 텍스트 */
  --text-muted: rgba(229, 211, 133, 0.4); /* 보조 텍스트 */
}
```

### 5.2 게임 프레임 스타일

- 외곽: 이중 테두리 (inner gold thin + outer dark thick)
- 모서리: 약간의 radius (8px) + 장식 요소 (코너 디테일)
- 배경: 미묘한 텍스처 또는 그라데이션 (순수 black이 아닌)
- 그림자: 내부 그림자로 깊이감

### 5.3 타이포그래피 (유지)

- **Spectral Bold**: 영문 제목/라벨
- **경기천년바탕 Bold**: 한국어 본문/제목
- **Kreon**: 숫자, 영문 보조
