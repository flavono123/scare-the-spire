# 슬서운 이야기 MVP

## 목표

슬레이 더 스파이어의 과거 밸런스 변경 이력을 "이야기"로 보여주는 사이트.
모든 카드/유물 데이터를 갖추되, 변경 이력이 있는 것들을 이야기로 큐레이션.

- e.g. "Corruption was originally an Uncommon card" → 클릭하면 변경 이력 표시

### 언어 정책

- 데이터는 영어 (게임 원문 기준)
- 서비스 UI는 한국어 우선, 영어 2순위
- 카드/유물 이름은 영어 원문 + 한국어 공식 번역 병기

## 기능

### 1. 이야기 피드 (메인)

- 변경 이력을 한 문장으로 요약한 "이야기" 카드를 피드로 노출
- 클릭 → 상세 (변경 이력 타임라인)
- 태그 분류 (너프/버프/리워크, 카드/유물, 캐릭터 클래스)

### 2. 카드/유물 탐색

- 모든 카드/유물을 탐색 가능 (변경 이력 유무 무관)
- 캐릭터별, 등급별 필터
- 변경 이력이 있으면 타임라인 표시

## 기술 스택

- **Frontend**: Next.js 15 (App Router) + shadcn/ui + Tailwind CSS
- **Data**: JSON 파일 (git 관리), 빌드 타임 로딩 (SSG)
- **Deploy**: Vercel (무료 tier, 자동 배포)
- **Lang**: TypeScript (strict)

## 데이터

- 전체 카드 (~200장) + 전체 유물 (~180개) 데이터 필수
- 변경 이력은 위키/패치노트 기반 수집
- → `prompts/SPEC.md` 참조

## MVP 범위

- 모든 카드/유물 데이터
- 이야기 피드 + 상세 (변경 이력 타임라인)
- 카드/유물 탐색 (필터)
- 반응형 (모바일 우선)
- Vercel 배포
