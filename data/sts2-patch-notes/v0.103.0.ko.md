## UI & UX

- 베타 브랜치 플레이어는 별도의 베타 전용 리더보드에 점수 등록
- 런 기록 화면에 스크롤바 추가
- 런 기록 화면에 배지 표시
- 멀티플레이어 일일 런 로딩 화면 정리
- 일일 리더보드에 점수 대신 층수, 배지 수, 시간 표시
- 멀티플레이어에서 골드 점수가 모든 플레이어 평균으로 통합
- 게임 오버 화면 리더보드 뷰 업데이트
- 일일 런 게임 오버 화면의 리더보드가 더 잘 맞도록 조정
- 리더보드에서 본인 이름이 파란색으로 강조 표시
- [gold]나선형 소용돌이[/gold] 이벤트 문구가 기본 타격 또는 수비만 [gold]인챈트[/gold]할 수 있음을 명시하도록 변경
- 컨트롤러에서 유물 인벤토리 바에 순환 탐색 추가
- [gold]수정구[/gold] 이벤트의 점술 버튼 단축키가 더 이상 덱 보기 버튼을 사용하지 않음
