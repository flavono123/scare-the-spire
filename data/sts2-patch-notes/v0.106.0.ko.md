## 버그 수정

### 일반

- 일일 도전 화면에 제출된 점수가 없는데도 세로 구분선이 표시되던 문제 수정.
- 두 언어 사이를 전환한 뒤 저장된 런을 이어 하면, 상단 바 일부 항목의 툴팁이 잘못된 언어로 표시될 수 있던 문제 수정.
- 엘리트 전투에서 승리해 포션 보상을 받은 뒤, 다음 전투에서 또 포션 보상을 받을 확률이 가끔 올라가던 문제 수정.
- 같은 버튼 입력으로 카드 검사 화면을 열고 강화 보기 버튼을 토글할 수 있던 문제 수정.
- 상단 보상 선택지를 비운 경우, 유물에서 보상 화면으로 가는 컨트롤러 탐색이 깨지던 문제 수정.
- 카드가 변화되는 중에 사용하려 하면 손패에 stuck 되던 문제 수정.
- 마우스가 화면에서 너무 낮은 위치에 있을 때 카드 선택 숫자 키가 작동하지 않던 문제 수정.
- deprecated 보스가 포함된 세이브 파일을 불러오면, 더 이상 같은 더블 보스가 굴러가지 않습니다.
- 게임 창이 포커스되지 않았을 때 컨트롤러 입력을 무시합니다.
- 시작 시 설정 저장 안정성 개선.
- 재사용된 카드가 재사용 사이에 파워가 사라진 경우에도 적절한 피해량을 표시합니다.
- Steam Cloud 충돌을 막기 위해 `settings.save` 클라우드 동기화를 중단했습니다.
- 카드 제거 보상을 마지막 보상으로 받을 때 배경이 흐리게 남던 문제 수정.
