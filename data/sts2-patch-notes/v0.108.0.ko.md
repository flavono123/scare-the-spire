- STS2.dll XML 문서 배포를 시작했습니다.
- 같은 모드를 사용하는 플레이어 사이에서 발생하던 모드 멀티플레이 "버전 불일치" 문제를 수정했습니다:

  - typename 대신 ModelId로 콘텐츠를 정렬합니다.
  - 정렬 시 더 많은 타이브레이커를 사용합니다.
  - `ModelIdSerializationCache`에서 역사 ID를 정렬합니다.
  - 게임플레이에 영향을 주지 않는 모델을 `ModelIdSerializationCache` 해시에서 제외합니다.
- 게임플레이에 영향을 주지 않는 모드 불일치 경고에 올바른 모드 목록이 기록됩니다.
