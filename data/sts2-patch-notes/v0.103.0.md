# Beta Patch Notes - v0.103.0

## Content & Balance

### Ancients

- Added new Neow relics:
  - **Neow's Bones** (new): "Upon pickup, gain 2 random Neow Relics. Add 1 random Curse to your Deck."
  - **Phial Holster** (new): "Upon pickup, gain 1 potion slot and procure 2 random potions."
  - **Winged Boots** (new): "You may ignore paths when choosing the next rooms to travel to 3 times."

### Ironclad

- **Not Yet** (new): "Rare - Skill - Cost 2 | Heal 10(13) HP. Exhaust."
- **Grapple** (deprecated): Removed from the game.
- **Colossus** (change): Rarity decreased from Rare -> Uncommon.

### Silent

- **Acrobatics** (change): Rarity increased from Common -> Uncommon.
- **Follow Through** (rework): "Common - Attack - Cost 1 | Deal 7(9) damage. If you have 5 or more other cards in your Hand, hits an additional time." (was "Uncommon - Attack - Cost 1 | Deal 6(8) damage to ALL enemies. If the last card you played this turn was a Skill, apply 1(2) Weak to ALL enemies.")
- **Memento Mori** (buff): Base damage increased from 8(10) -> 9(11).
- **Speedster** (buff): Damage increased from 1(2) -> 2. Upgrade changed from +1 damage -> Gains Innate.

### Regent

- **Regalite** (change): Now grants Block for any card created, not just Colorless cards.

### Enemies

- **Tunneling Twosome** (deprecated): Removed from the Hive.
- **Living Fog** (nerf): No longer increases the amount of bombs it spawns over time.
- **Skulking Colony** (nerf): HP decreased from 74(79) -> 70(75).
- **Byrdonis** (nerf): HP decreased from 91(99)-94(99) -> 81(90)-84(90).
- **Doormaker** (nerf): Overall damage numbers nerfed and moves rearranged. No longer applies debuffs. Gains less Strength.

## Art

- Added portrait art for the following cards: Adaptive Strike, Shadow Shield, Smokestack, Bad Luck, Outbreak
- Added new art for Nonupeipe Ancient background
- Added new art for the following Neow Ancient relics: Hefty Tablet, Neow's Talisman
- Updated art for Osty in character select screen

## UI & UX

- Players on the beta branch now submit their scores to a separate beta-only leaderboard
- Added a scrollbar to the Run History screen
- Badges now show up in Run History screen
- Cleaned up the screen when loading a multiplayer Daily Run
- Daily Leaderboards now show floor, badge count, and time instead of score
- In multiplayer games, gold score is averaged across all players for consistent score uploading
- Updated Leaderboard view in Game Over screen
- Leaderboards in Game Over screen now fit better in Dailies
- Your name on the Leaderboards is now highlighted blue
- Reworded Spiraling Whirlpool event to explicitly state it can only enchant a basic Strike or Defend
- Added wrapping navigation to Relic Inventory bar on controller
- Big divination button hotkey in Crystal Sphere event no longer uses deck view button on controller

## Writing

- Archaic Tooth relic now properly says "Transform a starter card into an Ancient version" instead of "with an Ancient version"
- Flak Cannon card description now highlights "Status" in gold
- Flanking card now properly specifies that it only doubles attack damage
- Iteration, Rocket Punch, and Trash to Treasure card descriptions now say "Status" instead of "Status card"
- Pael's Tooth relic now says "Cards devoured" instead of "Cards in the Pot"
- Updated translations for various languages
- Fixed translation errors for: German, Japanese, Russian
- Updated credits
