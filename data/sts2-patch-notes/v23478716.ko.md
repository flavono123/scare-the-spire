# 5월 버그 수정

베타 브랜치의 여러 버그 수정이 메인 브랜치로 넘어온 패치입니다.

## 버그 수정

### 일반

- 시작 시 설정 저장 안정성을 개선했습니다.
- 죽어가는 생명체에게 파워를 적용할 때 발생하던 크래시를 수정했습니다.
- 컨트롤러로 적을 대상으로 지정한 상태에서 카드 더미 화면을 열 때 발생하던 크래시를 수정했습니다.
- Linux에서 전투 중 UI 요소가 프레임 중간에 제거될 때 발생하던 크래시를 수정했습니다.
- Linux에서 ICU 라이브러리를 사용할 수 없을 때 발생하던 크래시를 수정했습니다.
- Linux에서 볼륨 키를 누르거나 alt-tab을 사용한 뒤 오디오가 음소거 상태로 남던 문제를 수정했습니다.
- Intel 620 GPU의 기본 렌더링 백엔드를 D3D12로 되돌렸습니다.
- 마우스 커서가 화면에서 너무 낮은 위치에 있을 때 숫자 키로 카드를 선택할 수 없던 문제를 수정했습니다.
- 카드 확인 화면을 열면서 같은 버튼 입력으로 강화 보기까지 토글할 수 있던 문제를 수정했습니다.
- 카드 확인 화면에서 강화 보기 단축키를 사용할 수 없던 문제를 수정했습니다.
- [gold:relic]판도라의 상자[/gold] 카드 보기 화면에서 카드 확인 버튼이 "accept" 단축키를 막던 문제를 수정했습니다.
- 컨트롤러로 [gold:card]염원[/gold] 카드를 사용할 때, 같은 버튼 입력으로 카드 격자의 카드가 곧바로 선택되던 문제를 수정했습니다.
- 최상단 보상 선택지를 비운 뒤 유물 목록에서 보상 화면으로 이동하는 컨트롤러 탐색이 깨지던 문제를 수정했습니다.
- 컨트롤러로 플레이 중일 때 게임이 "Controller Detected"와 "Mouse Detected" 사이를 빠르게 오가던 문제를 수정했습니다.
- 컨트롤러로 죽은 적을 대상으로 지정할 수 있던 문제를 수정했습니다.
- 상점에서 상단 바 밖으로 탐색할 수 없던 문제를 수정했습니다.
- 상점의 모든 유물과 무색 카드를 구매한 뒤 컨트롤러 탐색이 깨지던 문제를 수정했습니다.
- 유물이 두 줄일 때 컨트롤러로 휴식 장소 선택지를 탐색할 수 없던 문제를 수정했습니다.
- [gold]승천[/gold] 튜토리얼이 열린 상태에서 컨트롤러로 싱글 플레이어 런을 시작할 수 있던 문제를 수정했습니다.
- 설정이 저장된 뒤 새로 추가된 단축키의 입력 설정에서 발생하던 오류를 수정했습니다.
- 포션 제거 중 포션 슬롯에서 포커스가 해제될 때 발생하던 오류를 수정했습니다.
- [gold]타임라인[/gold] 화면의 화면 밖 역사 표시기로 인해 발생하던 오류를 수정했습니다.
- [gold]일일 도전[/gold] 리더보드 화면에서 제출된 점수가 없거나, 점수를 불러오는 중이거나, 불러오기에 실패했을 때 구분선이 표시되던 문제를 수정했습니다.
- [gold]재사용[/gold] 사이에 파워가 사라진 경우, 재사용된 카드가 올바른 피해량을 표시하지 않던 문제를 수정했습니다.
- [gold]밤의 공포[/gold] 변형자가 플레이어를 죽일 수 있던 문제를 수정했습니다.
- 삭제된 보스가 포함된 세이브 파일을 불러오면 더블 보스 전투에서 같은 보스가 두 번 나올 수 있던 문제를 수정했습니다.
- [gold:card]보물 지도[/gold] 퀘스트를 완료했을 때 여러 장을 가지고 있으면 모든 [gold:card]보물 지도[/gold] 카드가 제거되지 않던 문제를 수정했습니다.
- 디버그 파일 작성과 관련된 드문 소프트락을 수정했습니다.
- 정예 전투에서 승리하고 포션을 받은 뒤, 다음 포션 보상을 받을 확률이 감소하지 않고 오히려 증가할 수 있던 문제를 수정했습니다.

### [red]아이언클래드[/red]

- [gold:card]하사+[/gold]로 손에 들어온 이미 강화된 카드를 [gold:card]전투장비[/gold]로 선택할 수 있던 문제를 수정했습니다.

### [green]사일런트[/green]

- 엿보기 버튼으로 [gold:card]생존자[/gold] 카드의 버리기 선택을 건너뛸 수 있던 문제를 수정했습니다.

### [aqua]디펙트[/aqua]

- [gold:card]긁어내기[/gold] 카드가 [gold]에너지[/gold] 비용은 0이지만 [gold]별[/gold] 비용이 있는 카드를 버리던 문제를 수정했습니다.
- [gold:power]그림자 소모[/gold] 파워 툴팁의 간격 문제를 수정했습니다.

### [orange]리젠트[/orange]

- [gold:card]대령하라[/gold] 카드가 이제 [gold]단조[/gold] 이후가 아니라 이전에 기존 [gold:card]군주의 칼날[/gold]들을 손으로 가져오도록 수정했습니다.

### 포션 & 유물

- [gold:relic]장화[/gold] 유물이 [gold:power]불가침[/gold] 상태에서 [gold:card]불운[/gold] 저주로 받는 피해를 1이 아니라 5로 만들던 문제를 수정했습니다.
- [gold]불치병[/gold] 변형자의 최대 체력 손실로 [gold:relic]자가 형성 점토[/gold] 유물이 발동하던 문제를 수정했습니다.
- [gold:relic]속삭이는 귀걸이[/gold] 유물이 [gold:card]우주 먼지[/gold]를 0 [gold]별[/gold]이 아니라 올바른 양으로 재사용하도록 수정했습니다.

- [gold:monster]포식성 잭스프루트[/gold]가 [gold:power]반사[/gold] 파워로 죽을 때 발생하던 VFX 오류를 수정했습니다.
- [gold:monster]파지직봇[/gold]의 [gold:power]고전압[/gold] 파워 툴팁이 턴 시작이 아니라 턴 종료에 [gold:power]힘[/gold]을 얻는다고 올바르게 설명하도록 수정했습니다.
- [gold:monster]살아있는 안개[/gold]의 연기가 누락되던 문제를 수정했습니다.
- [gold:monster]의식의 신수[/gold]의 음악 전환이 기절 타이밍과 맞지 않던 문제를 수정했습니다.
- [gold]쇠약의 손길[/gold] 파워가 [gold:monster]의식의 신수[/gold]를 기절시킨 뒤에도 [gold:power]힘[/gold]을 얻게 만들던 문제를 수정했습니다.
- [gold:monster]폴짝이 도적[/gold]이 훔칠 카드가 없을 때 발생하던 소프트락을 수정했습니다.
- [gold:monster]투사된 공포[/gold] 환영을 해당 턴 동안 처치했을 때 [gold:card]어둠의 족쇄[/gold]로 받은 음수 [gold:power]힘[/gold]이 유지되던 문제를 수정했습니다.

- 설정 화면에서 창 크기를 조절할 때 다른 플레이어의 체력 바가 화면 밖으로 나가던 문제를 수정했습니다.
